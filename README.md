# DungeonCrawler
Programming 4 Dungeon Crawler code fix
Started by: Abraham Guernsey and Kane McCrory

Program Overview:
    - Runs a dungeon simulation game
    - Has various traps, monsters, and pitfalls that harm a player
    - Has various beneficial "treasures" that heal the player
    - Has a riddle/puzzle mechanic that can either harm or heal the player
        - Riddle harm is permanent and is different to player health
    - Run in a text editor
